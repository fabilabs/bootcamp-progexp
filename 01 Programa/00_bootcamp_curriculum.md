# Bootcamp Desarrollo Web Full Stack - 25 Sesiones
*120 horas totales: 105 horas formación + 15 horas proyecto*

## **MÓDULO 1: FUNDAMENTOS WEB (8 sesiones)**

### **Sesión 1 - Introducción y HTML Básico**
- **Tema:** Estructura básica de HTML, etiquetas fundamentales, semántica
- **Actividad:** Crear una página web personal con encabezados, párrafos, listas y enlaces

### **Sesión 2 - HTML Avanzado**
- **Tema:** Formularios, tablas, multimedia, elementos semánticos
- **Actividad:** Desarrollar un formulario de contacto completo con validaciones HTML

### **Sesión 3 - CSS Básico**
- **Tema:** Selectores, propiedades básicas, colores, tipografía, box model
- **Actividad:** Estilizar la página personal creada anteriormente

### **Sesión 4 - CSS Layout**
- **Tema:** Flexbox, posicionamiento, display
- **Actividad:** Crear un layout responsive con header, sidebar y footer

### **Sesión 5 - CSS Grid y Responsive**
- **Tema:** CSS Grid, media queries, mobile-first design
- **Actividad:** Desarrollar una galería de imágenes responsive

### **Sesión 6 - JavaScript Fundamentos**
- **Tema:** Variables, tipos de datos, operadores, estructuras de control
- **Actividad:** Crear una calculadora básica

### **Sesión 7 - JavaScript DOM**
- **Tema:** Manipulación del DOM, eventos, selección de elementos
- **Actividad:** Hacer interactiva la página personal con JavaScript

### **Sesión 8 - JavaScript Intermedio**
- **Tema:** Funciones, arrays, objetos, métodos de array
- **Actividad:** Desarrollar una lista de tareas (To-Do List) básica

## **MÓDULO 2: JAVASCRIPT AVANZADO (4 sesiones)**

### **Sesión 9 - ES6+ y Programación Asíncrona**
- **Tema:** Arrow functions, destructuring, async/await, promises
- **Actividad:** Crear una app del clima consumiendo una API pública

### **Sesión 10 - Local Storage y JSON**
- **Tema:** Almacenamiento local, manipulación de JSON, persistencia de datos
- **Actividad:** Mejorar el To-Do List con persistencia de datos

### **Sesión 11 - Fetch API y AJAX**
- **Tema:** Consumo de APIs REST, manejo de errores, estados de carga
- **Actividad:** Desarrollar un buscador de películas usando API externa

### **Sesión 12 - JavaScript Orientado a Objetos**
- **Tema:** Clases, herencia, módulos ES6
- **Actividad:** Refactorizar código anterior usando programación orientada a objetos

## **MÓDULO 3: REACT FUNDAMENTALS (5 sesiones)**

### **Sesión 13 - Introducción a React**
- **Tema:** JSX, componentes, props, instalación y configuración
- **Actividad:** Crear primeros componentes React básicos

### **Sesión 14 - Estado y Eventos en React**
- **Tema:** useState, manejo de eventos, formularios controlados
- **Actividad:** Convertir el To-Do List a React con estado

### **Sesión 15 - Props y Composición**
- **Tema:** Paso de props, composición de componentes, props children
- **Actividad:** Crear un sistema de componentes reutilizables

### **Sesión 16 - useEffect y Ciclo de Vida**
- **Tema:** useEffect, efectos secundarios, limpieza de efectos
- **Actividad:** Implementar funcionalidades de carga de datos y temporizadores

### **Sesión 17 - React Router**
- **Tema:** Navegación SPA, rutas, parámetros, rutas anidadas
- **Actividad:** Crear una aplicación multi-página con navegación

## **MÓDULO 4: NEXT.JS Y DESARROLLO AVANZADO (4 sesiones)**

### **Sesión 18 - Introducción a Next.js**
- **Tema:** SSR, SSG, file-based routing, instalación y configuración
- **Actividad:** Migrar la aplicación React a Next.js

### **Sesión 19 - API Routes en Next.js**
- **Tema:** Crear APIs backend, manejo de requests, middleware
- **Actividad:** Desarrollar endpoints API para la aplicación

### **Sesión 20 - Optimización y Performance**
- **Tema:** Image optimization, lazy loading, SEO básico, meta tags
- **Actividad:** Optimizar la aplicación para producción

### **Sesión 21 - Styling Avanzado**
- **Tema:** CSS Modules, Styled Components, Tailwind CSS
- **Actividad:** Implementar un sistema de diseño consistente

## **MÓDULO 5: BACKEND Y BASES DE DATOS (4 sesiones)**

### **Sesión 22 - Firebase Introducción**
- **Tema:** Firebase setup, Authentication, Firestore básico
- **Actividad:** Implementar sistema de autenticación de usuarios

### **Sesión 23 - Firestore Database**
- **Tema:** CRUD operations, queries, real-time listeners
- **Actividad:** Conectar la aplicación con base de datos en tiempo real

### **Sesión 24 - Firebase Hosting y Storage**
- **Tema:** Deploy, file upload, Storage rules, Security rules
- **Actividad:** Subir archivos y desplegar aplicación completa

### **Sesión 25 - Introducción a Supabase (Alternativa)**
- **Tema:** Supabase vs Firebase, PostgreSQL, Auth alternativa
- **Actividad:** Comparar y experimentar con Supabase como alternativa

---

## **PROYECTO FINAL (3.75 sesiones equivalentes)**
*15 horas distribuidas en las últimas sesiones*

**Desarrollo de una aplicación full-stack completa que integre:**
- Frontend React/Next.js responsive
- Autenticación de usuarios
- Base de datos con CRUD completo
- Consumo de APIs externas
- Deploy en producción

Las últimas 15 horas se pueden distribuir como:
- Últimos 45 minutos de las sesiones 22-25 para trabajo en proyecto
- Sesiones adicionales dedicadas exclusivamente al proyecto
- Tiempo para presentaciones y code review
